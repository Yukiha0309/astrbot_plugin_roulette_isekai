# AstrBot 异界战争轮盘

异界战争轮盘是从 `astrbot-plugin-roulette` 拆分出来的扩展玩法插件，包含异界道具、特殊道具、酒狐事件和独立配置。

## 指令

| 指令 | 说明 |
| --- | --- |
| `/异界轮盘创建` | 创建异界战争房间 |
| `/异界轮盘加入` | 加入等待中的异界战争房间 |
| `/异界退出房间` | 游戏开始前退出等待房间 |
| `/异界轮盘开始` | 房主或超级管理员开始游戏 |
| `/异界开自己` | 对自己开枪 |
| `/异界开 @用户` | 对指定存活玩家开枪 |
| `/异界听天由命` | 多人局随机选择存活玩家作为目标 |
| `/异界使用道具 道具名` | 使用道具 |
| `/异界使用酒狐委托` | 道具短指令，其他道具同理 |
| `/异界梭哈` | 查看梭哈确认提示，不直接执行 |
| `/异界梭哈 数量` | 最多使用指定数量，不跨弹仓 |
| `/异界梭哈 all` | 梭哈当前弹仓剩余全部子弹 |
| `/异界查看道具` | 查看自己的道具和特殊道具 |
| `/异界道具帮助 道具名` | 查看指定道具说明 |
| `/异界轮盘状态` | 查看当前房间状态 |
| `/异界替换特殊道具` | 接受待确认特殊道具 |
| `/异界放弃特殊道具` | 放弃待确认特殊道具 |
| `/异界死亡榜` | 查看本群死亡次数排行 |
| `/异界轮盘处决 @用户` | 房主或超级管理员处决挂机玩家 |
| `/异界轮盘结束` | 房主或超级管理员结束当前房间 |
| `/异界轮盘帮助` | 查看帮助 |

## v0.1.0

- 从 `astrbot-plugin-roulette` 拆分为独立插件。
- 保留异界战争已有玩法，不额外新增新玩法。
- 新增 `/异界道具帮助 道具名`。
- 新增 WebUI 分人数规则配置。
- 道具发放数量支持固定值，例如 `3`，也支持范围，例如 `1-3`。

## 人数规则

- 2 人局默认 5 血；第 3 轮开始发道具；每人随机 2-4 个；背包上限 8；前 3 轮最多 4 发。
- 3-4 人局默认 4 血；第 2 轮开始发道具；每人随机 2-3 个；背包上限 6；前 2 轮最多 5 发。
- 5-6 人局默认 3 血；第 1 轮开始发道具；每人随机 1-2 个；背包上限 4；前 2 轮最多 7 发。
- 7 人及以上默认 2 血；第 1 轮开始发道具；每人 1 个；背包上限 3；开局即最多 10 发。
- 5 人及以上开局随机 2 到 4 名玩家获得特殊道具 `梓的不死图腾`。

## 道具

| 道具 | 效果 |
| --- | --- |
| 凶吉签 | 查看当前子弹，凶为实弹，吉为空弹 |
| 绷带 | 回复 1 血并解除自己的流血状态 |
| 酒狐委托 | 酒狐往弹仓随机位置加入 1 发未知子弹 |
| 封膛符 | 目标下次行动可以使用道具，但不能真正开枪 |
| 锈蚀枪管 | 下一次实弹命中玩家时附加流血 |
| 加压弹簧 | 下一枪实弹伤害 +1，空弹则自己炸膛受 1 点伤害 |
| 隙间之手 | 随机偷取目标 1 个普通道具 |

## 特殊道具

| 道具 | 效果 |
| --- | --- |
| 梓的不死图腾 | 主动治疗自己或他人；受到致命伤害时自动保留 1 血 |
| 星之加护 | 下次受到大于 1 的实弹伤害时，抵消额外伤害 |
| 替罪签 | 被 `/异界听天由命` 抽中时自动消耗并重新随机目标 |

## 梭哈与加压弹簧

- 梭哈执行过程不会跨弹仓。
- 如果全部为空弹并打空当前弹仓，会进入下一弹仓轮且仍由当前玩家继续行动。
- 加压弹簧只影响梭哈第一枪。
- 第一枪空弹会炸膛扣血但继续梭哈。
- 第一枪实弹按加压伤害结算并切换回合。

## WebUI 配置

可在 AstrBot WebUI 中调整异界战争的分人数规则：

- 初始血量
- 第几轮开始发放道具
- 每次发放道具数量，支持 `3` 或 `1-3`
- 背包上限
- 前几轮限制子弹数
- 前期最多几发子弹
- 酒狐事件开关
- 酒狐事件触发概率
